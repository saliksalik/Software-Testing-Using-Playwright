# Module 3 - Authentication and Session

This module teaches secure API testing for login/session behavior.

## What you learn

- Auth handling concepts: API key, bearer token, JWT-style flow
- Reusing authenticated state with storageState
- Cookie and header behavior validation
- Access behavior for authenticated vs unauthenticated requests

## Run

```bash
npx playwright test "modules/Module 3 - Authentication and Session"
```
