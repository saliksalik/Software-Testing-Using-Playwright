# Module 4 - Integration and Mocking

This module covers service integration patterns and controlled mocking.

## What you learn

- API chaining in realistic flows (create/update/delete lifecycle)
- Network mocking for no-data and server-error scenarios
- Hybrid API + UI verification strategy

## Why this matters

Stable QA automation needs both real integrations and deterministic mocks.
