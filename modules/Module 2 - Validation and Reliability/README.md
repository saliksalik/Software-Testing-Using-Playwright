# Module 2 - Validation and Reliability

This module focuses on response validation depth and reliability checks.

## What you learn

- Schema validation with AJV and Zod
- Nested JSON assertion patterns
- Negative testing and error-case coverage
- Response time and latency expectations

## Why this matters

UI can look fine while API contracts are broken.
This module helps you catch backend issues early.

## Run

```bash
npx playwright test "modules/Module 2 - Validation and Reliability"
```
