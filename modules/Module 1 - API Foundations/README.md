# Module 1 - API Foundations

This module teaches the core API testing basics using Playwright.

## What you learn

- HTTP methods: GET, POST, PUT, PATCH, DELETE
- Request body, query parameters, and headers
- Basic response validation and status code checks
- Difference between full update (PUT) and partial update (PATCH)

## Practice flow

1. Start with static payload tests.
2. Move to dynamic payload and template-based requests.
3. Validate GET with headers and query parameters.
4. Finish with update and delete scenarios.

## Run

```bash
npx playwright test "modules/Module 1 - API Foundations"
```
