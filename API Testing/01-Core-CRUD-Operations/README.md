# Module 1: Core CRUD Operations

This module contains the core API request patterns for Create, Read, Update, and Delete workflows.

Contents:

- `5.1 POST Requests using Static API Request Body` — examples of hardcoded request payloads.
- `5.2 POST Requests using Static JSON File` — examples of external JSON payload files used for requests.
- `5.3 POST Requests using Dynamic API Request Body` — building request body payloads with runtime variables.
- `5.4 POST Requests using Dynamic JSON File` — using JSON templates and parameterized files.
- `5.5 GET API Request & Validating Response Headers` — fetching resources and validating headers.
- `5.6 Handling Query Parameters for Search & Filtering` — building requests with query params.
- `5.7 PUT API Request` — complete resource updates.
- `5.8 PATCH API Request` — partial resource updates.
- `5.9 DELETE API Request & Validating Deletion Logic` — delete operations and verification behavior.
