# Module 1: Core CRUD Operations

This module contains the core API request patterns for Create, Read, Update, and Delete workflows.

Contents:

-  POST Requests using Static API Request Body` — examples of hardcoded request payloads.
-  POST Requests using Static JSON File` — examples of external JSON payload files used for requests.
-  POST Requests using Dynamic API Request Body` — building request body payloads with runtime variables.
-  POST Requests using Dynamic JSON File` — using JSON templates and parameterized files.
-  GET API Request & Validating Response Headers` — fetching resources and validating headers.
-  Handling Query Parameters for Search & Filtering` — building requests with query params.
-  PUT API Request` — complete resource updates.
-  PATCH API Request` — partial resource updates.
-  DELETE API Request & Validating Deletion Logic` — delete operations and verification behavior.
