# Module 9 - Backend SQA Job Skills

This module is built from common backend SQA interview and job requirements.

## Skills covered

- API correctness (status, schema, required fields, errors)
- Business logic (discounts, totals, workflows)
- Authentication and authorization
- Database validation
- Integration testing
- Negative testing
- Performance basics
- Security checks (SQA-level)
- Observability checks

## Run module

```bash
npm run test:module-9
```

## Run one file example

```bash
npx playwright test test/module-9/api.correctness.status.schema.fields.errors.test.js
```

## Note

These tests are learning-focused and deterministic.
They simulate backend behavior patterns without depending on unstable external APIs.
